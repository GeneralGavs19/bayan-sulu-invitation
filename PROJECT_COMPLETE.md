## 🔑 Что нужно подготовить (15 минут)

### 1. MongoDB Atlas (бесплатно)
- Переходим на https://mongodb.com/cloud/atlas
- Создайте аккаунт и кластер (бесплатный)
- Скопируйте connection string
- **Результат**: `mongodb+srv://user:pass@cluster.mongodb.net/invitation_db`

### 2. Resend.com API (бесплатно)
- Переходим на https://resend.com
- Создайте аккаунт
- Получите API ключ
- **Результат**: `re_xxxxx...`

### 3. Создайте .env файл

`server/.env`:
```env
MONGODB_URI=mongodb+srv://user:password@cluster.mongodb.net/invitation_db
RESEND_API_KEY=re_xxxxx...
EMAIL_FROM=noreply@resend.dev
FRONTEND_URL=http://localhost:5173
PORT=3001
```

**Смотрите `SETUP_SERVICES.md` для подробных инструкций!**

## 🆘 Если что-то не работает

### Быстрая справка:

| Проблема | Решение |
|----------|---------|
| Ошибка: "Cannot find module" | Запустите `npm install` |
| MongoDB ошибка | Проверьте connection string и IP whitelist |
| Email не отправляется | Проверьте API ключ Resend |
| CORS ошибка | Обновите FRONTEND_URL в .env |
| Форма не отправляется | Откройте DevTools (F12) и смотрите консоль |

**Для подробного troubleshooting смотрите конец README.md**
